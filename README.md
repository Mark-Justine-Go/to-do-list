# To Do List

This project consists of 1 webpage with different modules showcasing a simple to do list web application.

Skills Demonstrated

- using modules to divide the codebase into smaller pieces for better readability and scalibity
- importing libraries from a third party source to make production faster
- using local storage as a temporary replacement for a real database
- using scripts to make bundling faster
- using different loaders to handle different file types

The following can be improved but I decided not to due to time constraint and because the main skills needed for the project has been touched upon already. The following can be easily fixed and does not break any functionality

- the web application is only using local storage which has a limit of 5mb, a real database can be easily integrated since handling the backend is a module which is separate from other functions
- adding task from different date reloads the website making you go back to the current date instead of staying in the date youre working on (it still functions correctly, just a hassle to go back to the date again)
- no settings
